공식 메뉴·신메뉴 자동 반영 기준

GitHub Pages 화면은 외부 사이트를 실시간 수집하지 않습니다. 함께 제공되는 GitHub Actions가 매일 공식 메뉴 페이지 변경을 감지해 GitHub Issue로 알려주며, 검토 후 HTML 데이터를 반영합니다.

공식 출처 우선브랜드 공식 메뉴와 공식 신메뉴 페이지에서 확인한 메뉴만 사용
NEW 자동 표시출시일로부터 90일 이내 메뉴는 자동으로 NEW 표시
기간 자동 관리판매 시작일 전 숨김, 종료일 이후 추천 자동 제외
30일 최신성 경고최종 확인 후 30일이 지나면 재확인 필요 표시
지점 검증 우선마곡나루 지점 판매 여부나 가격 미확인 시 매장 확인 필요 표시
가격 임의 입력 금지공식·지점 가격 미확인 항목은 총액 대신 매장 확인으로 표시
